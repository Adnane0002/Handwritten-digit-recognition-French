# **Title:** Handwritten Digit Recognition (**French**)

## **Contributors:**

* Adnane Lamnaouar
* Asmae Khiati

## **Summary:**

This project, conducted with passion as part of an academic initiative at École Centrale Méditerranée, explores and compares two approaches for handwritten digit recognition using deep neural networks. **The project is entirely in French.**

## **Achievements:**

* Implemented and trained a Multilayer Perceptron (MLP) and a Convolutional Neural Network (CNN) on the MNIST dataset.
* Evaluated the performance of MLP and CNN models on the MNIST dataset and on custom data.
* Developed an application for recognizing digits drawn in the Paint application.
* Implemented a system for recognizing digits from a camera image.
* Visualized the internal layers of the models, including feature maps for the CNN, providing insights into their functioning.

## **Results:**

* The CNN outperformed the MLP on the MNIST dataset and on custom data.
* The application was able to correctly identify digits drawn in Paint with high accuracy.
* The system was able to recognize digits from a camera image with a satisfactory success rate.
